## 015. 자기소개

> 신학기가 시작되고, 아이들이 돌아가면서 자기소개를 하기로 했습니다. <br><br>
> 만약 입력으로 `김다정`이라는 이름이 주어지면 "안녕하세요. 저는 김다정입니다."라고 출력하게 해주세요.

```md
입출력

입력 : 김다정
출력 : 안녕하세요. 저는 김다정입니다.
```

<br>

### _💡 풀이_

```js
const input = prompt("이름을 입력하세요.");

function solution(name) {
  console.log(`안녕하세요. 저는 ${name}입니다.`);
}

solution(input);
```
